Program Coordinator

Compensation: Paid · Timeline: Ongoing · Git Enabled: Yes

Orchestrate schedules, milestones, and cross-team communication for the POC. You’ll coordinate planning, run status reviews, and ensure documentation and stakeholder updates are timely and accurate.

Key Responsibilities

  • Maintain project plans, milestones, dependencies, and risks.
  • Facilitate weekly standups, reviews, and cross-team syncs.
  • Drive documentation updates and publish status dashboards.

Minimum Qualifications

  • Project coordination experience and excellent communication.
  • Comfort with GitHub boards, docs, and lightweight reporting.
