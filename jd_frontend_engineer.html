Frontend Engineer (Developer Tools)

Compensation: Paid · Timeline: 3–5 months · Git Enabled: Yes

Create intuitive web UIs for prompts, runs, dashboards, and artifact exploration. You’ll design components that make evaluation workflows discoverable and efficient, integrate with backend APIs, and surface metrics and logs in accessible ways for researchers and engineers.

Key Responsibilities

  • Build responsive UI components for experiment setup, status, and results.
  • Integrate charts and tables for benchmark metrics and trends.
  • Collaborate on UX for sandboxed execution and error visibility.

Minimum Qualifications

  • Experience with modern web stack (HTML/CSS/JS; a framework is a plus).
  • REST API consumption, auth patterns, and state management.
  • Accessibility and responsive design awareness.

Preferred Qualifications

  • Data visualization (e.g., D3/Chart.js) for metrics dashboards.
  • Form builders and validation for prompt/config inputs.
  • Performance profiling and Lighthouse metrics improvements.
