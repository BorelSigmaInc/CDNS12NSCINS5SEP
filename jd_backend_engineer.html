Backend Engineer (Evaluation APIs)

Compensation: Paid · Timeline: 3–5 months · Git Enabled: Yes

Build secure, reliable services for submitting evaluation jobs, collecting results, and serving analytics. You will design APIs, storage schemas, and background processing that integrate seamlessly with training and benchmarking workflows while enforcing sandbox and security constraints.

Key Responsibilities

  • Design REST/JSON APIs for job submission, status tracking, and results retrieval.
  • Implement idempotent workers and queues for evaluation execution.
  • Model storage for runs, metrics, artifacts, and lineage metadata.
  • Enforce authentication/authorization and rate-limits; add audit logs.

Minimum Qualifications

  • Experience with a backend framework (FastAPI/Flask/Django/Node) and SQL/NoSQL.
  • Background jobs/queues (Celery/RQ) and object storage familiarity.
  • Understanding of containerized deployments and CI.

Preferred Qualifications

  • Knowledge of secure coding and OWASP guidelines.
  • Metrics/observability and structured logging for APIs.
  • Experience with schema migrations and data retention policies.
