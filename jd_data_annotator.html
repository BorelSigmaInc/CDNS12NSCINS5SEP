Data Annotator (Intern)

Compensation: Unpaid · Timeline: Ongoing · Git Enabled: Yes

Label prompts, code, and outcomes to support dataset curation and quality control. You will follow detailed guidelines, ensure consistency, and surface edge cases that inform model improvements and test design.

Key Responsibilities

  • Annotate prompts and generated code with correctness and category labels.
  • Apply quality checklists; escalate ambiguous cases and policy violations.
  • Contribute feedback to improve labeling guides and examples.

Minimum Qualifications

  • Attention to detail, ability to follow structured instructions.
  • Basic Python familiarity a plus; willingness to learn evaluation criteria.

Methodology & Practices

  • Double-blind checks and inter-annotator agreement monitoring.
  • Versioned label sets and audit trails for corrections.

Suggested Reading

  • HumanEval tasks and solutions
  • Pass@k metric context
